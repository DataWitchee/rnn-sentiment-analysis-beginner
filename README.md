# rnn-sentiment-analysis-beginner
A beginner-friendly sentiment analysis project using Simple RNN with TensorFlow/Keras. Demonstrates text preprocessing, tokenization, and sequence modeling on a small custom dataset.


This project implements a basic Recurrent Neural Network (RNN) for sentiment classification using TensorFlow and Keras.
The goal of this project is to understand how sequential models process text data, including tokenization, padding, and embeddings.

##this project is not industry level , it is simply beginner level if someone wants to understand working of rnn using simple data set.
In this project i have taken 30 statements 15 positive and 15 negative and trained a simple RNN model on it for sentiment analysis.
